This project is a Weighted Scoring Model designed to solve "analysis paralysis" when faced with multiple competing career paths. By quantifying subjective and objective criteria, the tool provides a mathematical foundation for choosing the option with the highest probability of success and satisfaction.
​How it Works
​The decision matrix operates on a Weighted Multiplier logic:
​Variable Weighting: Each criterion is assigned a weight based on its priority (e.g., Job Market = 5, Pay = 3, Schooling = 1).
​Raw Scoring: Each career path is given a raw score for each category.
​Weighted Subtotaling: The raw score is multiplied by the weight.
​Formula: (Raw Score * Weight) = Subtotal
​Grand Total: The sum of all subtotals determines the winning option.
​Evaluation Criteria
​Economic Factors: Pay and Job Market demand.
​Personal Factors: Enjoyment and My Talent (Aptitude).
​Investment Factors: Schooling (Educational cost/time).
​Technical Highlights
​Dynamic Weighting: The spreadsheet is built to allow users to change weights based on their personal values, which automatically updates the entire ranking system.
​UI Design: Used color-blocking to separate criteria for better visual clarity and readability.
​Decision Result: In the current configuration, Engineering emerged as the optimal path, balancing a strong job market with high talent alignment.
​Use Cases
​While this version focuses on career decisions, the underlying logic is applicable to:
​Business: Vendor selection or project prioritization.
​Product: Feature roadmapping.
​Personal Finance: Real estate or investment comparisons.
