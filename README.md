Project Overview
вЂ‹This project is a Weighted Scoring Model designed to solve "analysis paralysis" when faced with multiple competing career paths. By quantifying subjective and objective criteria, the tool provides a mathematical foundation for choosing the option with the highest probability of success and satisfaction.
вЂ‹How it Works
вЂ‹The decision matrix operates on a Weighted Multiplier logic:
вЂ‹Variable Weighting: Each criterion is assigned a weight based on its priority (e.g., Job Market = 5, Pay = 3, Schooling = 1).
вЂ‹Raw Scoring: Each career path is given a raw score for each category.
вЂ‹Weighted Subtotaling: The raw score is multiplied by the weight.
вЂ‹Formula: (Raw Score * Weight) = Subtotal
вЂ‹Grand Total: The sum of all subtotals determines the winning option.
вЂ‹Evaluation Criteria
вЂ‹Economic Factors: Pay and Job Market demand.
вЂ‹Personal Factors: Enjoyment and My Talent (Aptitude).
вЂ‹Investment Factors: Schooling (Educational cost/time).
вЂ‹Technical Highlights
вЂ‹Dynamic Weighting: The spreadsheet is built to allow users to change weights based on their personal values, which automatically updates the entire ranking system.
вЂ‹UI Design: Used color-blocking to separate criteria for better visual clarity and readability.
вЂ‹Decision Result: In the current configuration, Engineering emerged as the optimal path, balancing a strong job market with high talent alignment.
